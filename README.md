# love.leorza.net

Página estática temática (love.leorza.net) hospedada no servidor pessoal.

## Stack

Site estático — HTML/CSS/JS puro.

## Como usar

```bash
python -m http.server 8000
```

Ou abra `index.html` diretamente no navegador.

## Conteúdo

- `index.html` — página principal
- `love.png`, `howiwant.png` — imagens
- `favicon.ico` / `favicon.svg`
